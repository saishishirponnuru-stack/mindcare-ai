# 🧠 MindCare AI

MindCare AI is an AI-powered mental wellness platform that helps users track emotions, journal their thoughts, and gain personalized wellbeing insights.

## 🚀 Live Demo

https://mindcare-ai-by-shishir.streamlit.app

## ✨ Features

- 💬 AI Emotional Support Chatbot
- ❤️ Wellness Score Analysis
- 📓 Daily Journaling
- 📊 Mood Analytics Dashboard
- 🧠 Emotional Insights
- 🔥 Admin Dashboard

- Google Gemini AI
